## Was ist FluxTurn?

FluxTurn ist eine Open-Source-Plattform zur Workflow-Automatisierung, mit der Sie Apps verbinden, Prozesse automatisieren und KI-gestutzte Workflows erstellen konnen -- alles uber einen visuellen Editor oder naturliche Sprache.

## Hauptfunktionen

- **Visueller Workflow-Editor** - Erstellen Sie komplexe Automatisierungen per Drag-and-Drop
- **KI-gestutzte Workflow-Generierung** - Beschreiben Sie Ihre Anforderungen in naturlicher Sprache, die KI generiert den Workflow
- **Uber 80 Konnektoren** - CRM, Marketing, KI/ML, Kommunikationstools und mehr
- **Self-Hosting** - Einfaches Deployment mit `docker compose up`
- **Echtzeit-Ausfuhrung** - Echtzeit-Uberwachung uber WebSocket
- **Benutzerdefinierter Code** - Unterstutzung fur JavaScript/Python-Ausfuhrung

## Schnellstart

```bash
git clone https://github.com/fluxturn/fluxturn.git
cd fluxturn
docker compose --env-file .env.docker up
```

Frontend: http://localhost:5185
Backend-API: http://localhost:5005

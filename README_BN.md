## FluxTurn কী?

FluxTurn হল একটি **প্রোডাকশন-রেডি, ওপেন-সোর্স ওয়ার্কফ্লো অটোমেশন প্ল্যাটফর্ম** যা ধারণা এবং বাস্তবায়নের মধ্যে ব্যবধান পূরণ করে। ডেভেলপার, DevOps টিম এবং প্রযুক্তিগত ব্যবহারকারীদের জন্য নির্মিত, FluxTurn AI-চালিত ওয়ার্কফ্লো জেনারেশনের শক্তিকে একটি পরিশীলিত ভিজ্যুয়াল বিল্ডারের সাথে একত্রিত করে আপনাকে ঘন্টার পরিবর্তে সেকেন্ডে জটিল প্রক্রিয়া স্বয়ংক্রিয় করতে সাহায্য করে।

ঐতিহ্যবাহী অটোমেশন টুলগুলির বিপরীতে যা ব্যাপক কনফিগারেশন প্রয়োজন বা লো-কোড প্ল্যাটফর্ম যা নমনীয়তা ত্যাগ করে, FluxTurn আপনাকে দুই জগতের সেরা দেয়: প্রাকৃতিক ভাষা ওয়ার্কফ্লো জেনারেশনের গতি এবং একটি ভিজ্যুয়াল নোড-ভিত্তিক এডিটরের নির্ভুলতা।

### এটি কিভাবে কাজ করে

1. **আপনার ওয়ার্কফ্লো বর্ণনা করুন** -- FluxTurn-কে সাধারণ ইংরেজিতে বলুন আপনি কী স্বয়ংক্রিয় করতে চান
2. **AI ফ্লো তৈরি করে** -- আমাদের AI এজেন্ট আপনার প্রয়োজনীয়তা বিশ্লেষণ করে এবং সঠিক কানেক্টর সহ একটি সম্পূর্ণ ওয়ার্কফ্লো তৈরি করে
3. **ভিজ্যুয়াল রিফাইনমেন্ট** -- আমাদের ReactFlow-চালিত ক্যানভাস ব্যবহার করে তৈরি ওয়ার্কফ্লো ফাইন-টিউন করুন
4. **ডিপ্লয় এবং মনিটর করুন** -- বিস্তারিত লগিং এবং WebSocket-ভিত্তিক মনিটরিং সহ রিয়েল-টাইমে ওয়ার্কফ্লো এক্সিকিউট করুন

### মূল সক্ষমতা

- **🤖 AI ওয়ার্কফ্লো জেনারেশন** -- সাধারণ ইংরেজিতে বর্ণনা করুন আপনি কী চান, সঠিক ত্রুটি পরিচালনা এবং সর্বোত্তম অনুশীলন সহ একটি কার্যকর ওয়ার্কফ্লো পান
- **🎨 ভিজ্যুয়াল ওয়ার্কফ্লো বিল্ডার** -- রিয়েল-টাইম যাচাইকরণ সহ ReactFlow দ্বারা চালিত ড্র্যাগ-এন্ড-ড্রপ ইন্টারফেস
- **🔌 ১২০+ প্রি-বিল্ট কানেক্টর** -- Slack, Gmail, Shopify, HubSpot, Jira, Stripe, OpenAI, Anthropic, এবং আরও অনেক কিছু
- **⚡ রিয়েল-টাইম এক্সিকিউশন** -- বিস্তারিত লগ, WebSocket আপডেট এবং পারফরম্যান্স মেট্রিক্স সহ ওয়ার্কফ্লো চলতে দেখুন
- **🏠 সেল্ফ-হোস্টেড এবং গোপনীয়তা-প্রথম** -- Docker সহ আপনার নিজের অবকাঠামোতে চালান, সম্পূর্ণ ডেটা নিয়ন্ত্রণ
- **🌍 মাল্টি-ল্যাঙ্গুয়েজ সাপোর্ট** -- ইংরেজি, জাপানি, চীনা, কোরিয়ান, স্প্যানিশ এবং আরও অনেক ভাষা সহ ১৭টি ভাষা
- **🔄 প্রোডাকশন-রেডি** -- এন্টারপ্রাইজ-স্কেল ডিপ্লয়মেন্টের জন্য NestJS, PostgreSQL, Redis এবং Qdrant দিয়ে নির্মিত

## আমরা কী সমস্যা সমাধান করি

### অটোমেশন দ্বিধা

আধুনিক টিমগুলি একটি গুরুত্বপূর্ণ চ্যালেঞ্জের মুখোমুখি: **অটোমেশন অপরিহার্য কিন্তু সময়সাপেক্ষ**। টুলগুলির মধ্যে ইন্টিগ্রেশন তৈরি করা, ত্রুটি পরিচালনা করা এবং ওয়ার্কফ্লো বজায় রাখার জন্য উল্লেখযোগ্য প্রকৌশল সংস্থান প্রয়োজন।

**আমরা যে সাধারণ সমস্যাগুলি সমাধান করি:**

- ❌ **ম্যানুয়াল ইন্টিগ্রেশন নরক** -- বিভিন্ন API সংযুক্ত করতে কাস্টম স্ক্রিপ্ট লিখতে ঘন্টা বা দিন লাগে
- ❌ **ব্যয়বহুল SaaS লক-ইন** -- বাণিজ্যিক অটোমেশন টুলগুলি প্রতি ওয়ার্কফ্লো এক্সিকিউশন বা ব্যবহারকারী সিটের জন্য চার্জ করে
- ❌ **সীমিত নমনীয়তা** -- লো-কোড প্ল্যাটফর্মগুলি শুরু করা সহজ কিন্তু জটিল ব্যবহারের ক্ষেত্রে কাস্টমাইজ করা কঠিন
- ❌ **ভেন্ডর ডিপেন্ডেন্সি** -- ক্লাউড-শুধুমাত্র সমাধান মানে আপনি আপনার অটোমেশন লজিক বা ডেটার মালিক নন
- ❌ **খাড়া শেখার বক্ররেখা** -- ঐতিহ্যবাহী ওয়ার্কফ্লো ইঞ্জিনগুলি সেট আপ করতে গভীর প্রযুক্তিগত জ্ঞান প্রয়োজন

### FluxTurn-এর সমাধান

✅ **AI-চালিত গতি** -- ঘন্টার পরিবর্তে সেকেন্ডে ধারণাকে কার্যকর ওয়ার্কফ্লোতে রূপান্তরিত করুন
✅ **ওপেন সোর্স স্বাধীনতা** -- কোন ভেন্ডর লক-ইন নেই, প্রতি-এক্সিকিউশন ফি নেই, আপনার কোডের উপর সম্পূর্ণ নিয়ন্ত্রণ
✅ **সেল্ফ-হোস্টেড গোপনীয়তা** -- সংবেদনশীল ডেটা এবং ওয়ার্কফ্লো আপনার অবকাঠামোতে রাখুন
✅ **ডেভেলপার-ফ্রেন্ডলি** -- সম্পূর্ণ API অ্যাক্সেস, এক্সটেন্সিবল কানেক্টর সিস্টেম, TypeScript কোডবেস
✅ **ভিজ্যুয়াল + কোড** -- AI জেনারেশন দিয়ে শুরু করুন, ভিজ্যুয়ালি রিফাইন করুন, প্রয়োজনে কোড হিসাবে এক্সপোর্ট করুন

### FluxTurn-কে অনন্য করে তোলে কী?

1. **AI-ফার্স্ট ডিজাইন** -- নেটিভ AI ওয়ার্কফ্লো জেনারেশন এবং প্রাকৃতিক ভাষা বোঝার সাথে একমাত্র ওয়ার্কফ্লো প্ল্যাটফর্ম
2. **আধুনিক টেক স্ট্যাক** -- React 19, NestJS, PostgreSQL, Redis, Qdrant -- ২০২৫ এবং তার পরেও জন্য নির্মিত
3. **ডেভেলপার এক্সপেরিয়েন্স** -- ক্লিন TypeScript কোডবেস, এক্সটেন্সিবল আর্কিটেকচার, বিস্তৃত API
4. **সত্যিকারের ওপেন সোর্স** -- Apache 2.0 লাইসেন্স, কোন "fair-code" সীমাবদ্ধতা নেই, কমিউনিটি-চালিত উন্নয়ন
5. **মাল্টি-মোডাল ইনপুট** -- প্রাকৃতিক ভাষা বা ভিজ্যুয়াল বিল্ডার বা API -- আপনার টিমের জন্য কী কাজ করে তা চয়ন করুন

## দ্রুত শুরু

### Docker (প্রস্তাবিত)

প্রজেক্ট রুট থেকে এই কমান্ডগুলি চালান:

```bash
git clone https://github.com/fluxturn/fluxturn.git
cd fluxturn
cp backend/.env.example backend/.env
# আপনার ডাটাবেস ক্রেডেনশিয়াল এবং JWT সিক্রেট দিয়ে backend/.env এডিট করুন
docker compose up -d
```

এটুকুই! `http://localhost:5185` এ অ্যাপ এবং `http://localhost:5005` এ API অ্যাক্সেস করুন।

### ম্যানুয়াল সেটআপ

**পূর্বশর্ত:** Node.js 18+, PostgreSQL 14+, Redis 7+

```bash
# ক্লোন করুন
git clone https://github.com/fluxturn/fluxturn.git
cd fluxturn

# ব্যাকএন্ড
cd backend
cp .env.example .env    # আপনার কনফিগারেশন দিয়ে .env এডিট করুন
npm install
npm run start:dev

# ফ্রন্টএন্ড (একটি নতুন টার্মিনালে)
cd frontend
cp .env.example .env
npm install
npm run dev
```

## 🚀 FluxTurn-এ অবদান রাখার কারণ কী?

FluxTurn শুধুমাত্র আরেকটি ওপেন-সোর্স প্রকল্প নয় -- এটি অত্যাধুনিক প্রযুক্তির সাথে কাজ করার একটি সুযোগ যা বিশ্বব্যাপী ডেভেলপারদের জন্য প্রকৃত সমস্যা সমাধান করে এমন কিছু তৈরি করার সময়।

### আপনি কী পাবেন

**📚 আধুনিক টেক স্ট্যাক শিখুন**
- **React 19** -- সার্ভার কম্পোনেন্ট সহ সর্বশেষ React ফিচার
- **NestJS** -- এন্টারপ্রাইজ দ্বারা ব্যবহৃত পেশাদার ব্যাকএন্ড ফ্রেমওয়ার্ক
- **LangChain** -- AI/ML ইন্টিগ্রেশন এবং এজেন্ট অর্কেস্ট্রেশন
- **ভেক্টর ডাটাবেস** -- শব্দার্থিক অনুসন্ধানের জন্য Qdrant নিয়ে কাজ করুন
- **ReactFlow** -- ইন্টারেক্টিভ নোড-ভিত্তিক UI তৈরি করুন
- **রিয়েল-টাইম সিস্টেম** -- WebSocket, Redis এবং ইভেন্ট-চালিত আর্কিটেকচার

**💼 আপনার পোর্টফোলিও তৈরি করুন**
- একটি **প্রোডাকশন-রেডি** প্ল্যাটফর্মে অবদান রাখুন যা প্রকৃত কোম্পানিগুলি ব্যবহার করে
- এমন ফিচারগুলিতে কাজ করুন যা আপনার GitHub প্রোফাইলে প্রদর্শিত হয়
- আমাদের অবদানকারী হল অফ ফেমে স্বীকৃতি পান
- **ওয়ার্কফ্লো অটোমেশন** এবং **AI ইন্টিগ্রেশন** এ দক্ষতা তৈরি করুন -- ২০২৬ সালে অত্যন্ত মূল্যবান দক্ষতা

**🤝 একটি ক্রমবর্ধমান কমিউনিটিতে যোগ দিন**
- সারা বিশ্ব থেকে ডেভেলপারদের সাথে সংযুক্ত হন
- অভিজ্ঞ রক্ষণাবেক্ষণকারীদের কাছ থেকে কোড রিভিউ পান
- সফটওয়্যার আর্কিটেকচারে সর্বোত্তম অনুশীলন শিখুন
- প্রযুক্তিগত আলোচনা এবং ডিজাইন সিদ্ধান্তে অংশগ্রহণ করুন

**🎯 প্রকৃত প্রভাব ফেলুন**
- আপনার কোড হাজার হাজার ডেভেলপারকে তাদের ওয়ার্কফ্লো স্বয়ংক্রিয় করতে সাহায্য করবে

## 🎯 দ্রুত অবদান গাইড

### ধাপ ১: আপনার পরিবেশ সেট আপ করুন

```bash
# GitHub-এ রিপোজিটরি ফর্ক করুন, তারপর আপনার ফর্ক ক্লোন করুন
git clone https://github.com/YOUR_USERNAME/fluxturn.git
cd fluxturn

# Docker দিয়ে শুরু করুন (সবচেয়ে সহজ উপায়)
cp backend/.env.example backend/.env
docker compose up -d

# অ্যাপ অ্যাক্সেস করুন
# ফ্রন্টএন্ড: http://localhost:5185
# ব্যাকএন্ড API: http://localhost:5005
```

**এটুকুই!** আপনি স্থানীয়ভাবে FluxTurn চালাচ্ছেন।

### ধাপ ৩: আপনার পরিবর্তন করুন

```bash
# একটি নতুন ব্র্যাঞ্চ তৈরি করুন
git checkout -b feature/your-feature-name

# আপনার পরিবর্তন করুন
# - ফ্রন্টএন্ড কোড: /frontend/src
# - ব্যাকএন্ড কোড: /backend/src
# - কানেক্টর: /backend/src/modules/fluxturn/connectors

# আপনার পরিবর্তন পরীক্ষা করুন
npm run test

# একটি স্পষ্ট বার্তা সহ কমিট করুন
git commit -m "feat: add new connector for Notion API"
```

### ধাপ ৪: আপনার Pull Request জমা দিন

```bash
# আপনার ফর্কে পুশ করুন
git push origin feature/your-feature-name

# GitHub-এ একটি PR খুলুন
# - আপনি কী পরিবর্তন করেছেন এবং কেন তা বর্ণনা করুন
# - কোন সম্পর্কিত ইস্যুতে লিঙ্ক করুন
# - এটি একটি UI পরিবর্তন হলে স্ক্রিনশট যোগ করুন
```

**এরপর কি ঘটে?**
- ✅ আপনার PR-এ স্বয়ংক্রিয় টেস্ট চলে
- 👀 একজন রক্ষণাবেক্ষণকারী আপনার কোড পর্যালোচনা করেন (সাধারণত ২৪-৪৮ ঘন্টার মধ্যে)
- 💬 আমরা পরিবর্তন বা উন্নতি সাজেস্ট করতে পারি
- 🎉 একবার অনুমোদিত হলে, আপনার কোড মার্জ হয়ে যায়!

## O que e o FluxTurn?

FluxTurn e uma plataforma open source de automacao de workflows que permite conectar aplicativos, automatizar processos e criar workflows com IA -- tudo atraves de um editor visual ou linguagem natural.

## Principais Funcionalidades

- **Editor Visual de Workflows** - Construa automacoes complexas com arrastar e soltar
- **Geracao de Workflows com IA** - Descreva o que precisa em linguagem natural, a IA gera o workflow
- **80+ Conectores** - CRM, marketing, IA/ML, ferramentas de comunicacao e mais
- **Self-hosting** - Deploy facil com `docker compose up`
- **Execucao em Tempo Real** - Monitoramento em tempo real via WebSocket
- **Codigo Personalizado** - Suporte para execucao JavaScript/Python

## Inicio Rapido

```bash
git clone https://github.com/fluxturn/fluxturn.git
cd fluxturn
docker compose --env-file .env.docker up
```

Frontend: http://localhost:5185
API Backend: http://localhost:5005

## Qu'est-ce que FluxTurn ?

FluxTurn est une plateforme open source d'automatisation de workflows qui vous permet de connecter des applications, d'automatiser des processus et de creer des workflows propulses par l'IA, le tout via un editeur visuel ou en langage naturel.

## Fonctionnalites principales

- **Editeur visuel de workflows** - Construisez des automatisations complexes par glisser-deposer
- **Generation de workflows par IA** - Decrivez vos besoins en langage naturel, l'IA genere le workflow
- **Plus de 80 connecteurs** - CRM, marketing, IA/ML, outils de communication et plus
- **Auto-hebergement** - Deployez facilement avec `docker compose up`
- **Execution en temps reel** - Surveillance en temps reel via WebSocket
- **Code personnalise** - Support de l'execution JavaScript/Python

## Demarrage rapide

```bash
git clone https://github.com/fluxturn/fluxturn.git
cd fluxturn
docker compose --env-file .env.docker up
```

Frontend : http://localhost:5185
API Backend : http://localhost:5005
